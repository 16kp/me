### Setup
* Install create-react-app
    ```
    npm install create-react-app
    ```

* Install material-ui core, material-ui icons
    ```
    npm install @material-ui/core
    npm install @material-ui/icons
    ```


### Deploy on gh-pages
* build
    ```
    npm run predeploy
    ```

* deploy built changes
    ```
    npm run deploy
    ```
